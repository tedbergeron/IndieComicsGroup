Keyboard macro

With the keyboard macro kbd:[] we can include nicely formatted keyboard shortcuts.

Requires these attribute options
// We must enable experimental attribute.
:experimental:
// Define unicode for Apple Command key.
:commandkey: &#8984;
Keyboard macro syntax

Press ⌘+1 or Ctrl+1 to access the Project view.

To zoom out press Ctrl+-.

Find files with Ctrl+Alt+N or ⌘+Shift+N.

Button macro syntax

Press the OK button when you are finished.

Select a file in the file navigator and click Open.

Menu macro syntax

To save the file, select File ▸ Save.

Select View ▸ Zoom ▸ Reset to reset the zoom level to the default setting.

Reality Competition to Find New Talent

I’ve been thinking about ways to promote indie comics. Perhaps if we could put together our own "Idol" competition for comics. We could get a lot of publicity for the competitors. It would give us a chance to talk about what we like and what we don’t like. We could establish some minimum requirements. These standards would help define when the aspiring maker was ready to compete. I think it would inspire comic book makers.

Preparing for Free Comic Book Day

FCBD Well 2012 is nearly here and already the big publishers have got their full line up for Free Comic Book Day coming up Saturday May, 5th 2012. The Future of Publishing

By Ted Bergeron I’m trying to imagine what the future of comics will be. So I’m trying to create a 'blueprint' for making comics so that they still could be printed. Then take those files and make mobile apps, color e-books, web comics… every method of sharing comics digitally. The idea is books could be printed, if anyone wanted to pay a lot for one, but mostly they’d be shared electronically.

The Beginning

Welcome to the beginning. I’ve created this website with the intention of helping others make comics and share comics. I teach a making comic books class at my local Community College. I find that many people have a desire to create stories but not very many people know what all goes into making comics. After they’ve made some comics the next question is usually how do I get published? Which generally translates into how do I become rich and famous making comics.
